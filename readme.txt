== Description ==

This plugin add a widget to the widgets zone. 

You can configure the widget for show custom category posts, 
custom post types posts and what you image with that. 

Simple and easy to use. 

Thanks for using


Features (en_EN):

*   You can show last posts on a category (slug name or id)
*   You can show last posts custom post types. (post type name)
*   You can show multiple posts as many as you want (custom number)
*   You can show multiple post type with seperating commas.



Özellikler (tr_TR):

*   Kategorilere göre son yazılar gösterebilrisiniz. (slug name veya id)
*   Özel yazı tipine göre son yazılar gösterebilirsiniz.
*   Çoklu özel yazı tipine göre son yazılar gösterebilirsiniz.
